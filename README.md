# Цель проекта

Цель проекта — "демонстрация навыка css 4 уровня для роли "Эксперт"".

## Как запустить проект

1. Установите зависимости: `npm install`.
2. Для локальной разработки запустите `npm run dev` и перейдите по адресу из консоли (по умолчанию http://localhost:5173).
3. Для проверки production-сборки выполните `npm run build`, затем `npm run preview`.
4. Дополнительно доступны команды `npm run lint` (проверка качества кода) и `npm run deploy` (публикация содержимого `dist` через gh-pages).
